# Linux
## ■ Linuxとは
OSの一種でオープンソースのOS。  
リーナスという天才のアイディアと無数のエンジニアのボランティア精神がいつの間にか世界に広がっている。  

## ■ Linuxコマンド集

* 現在作業中のディレクトリを表示  
```
$ pwd
```  

* ディレクトリの中身を表示  
```
$ ls
```  

 * ホームディレクトリへ移動  
```
$ cd
```  

 * 指定したディレクトリに移動  
```
$ cd <移動先のディレクトリ>
```  

* ファイルを移動  
```
$ mv <移動させたいファイル> <移動後のファイル名>
```  

* ファイルをコピー  
```
$ cp <コピーしたいファイルのディレクトリ/ファイル名>
```
