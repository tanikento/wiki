* [Vue.js_Rails_Docker](https://tanikento.github.io/wiki/環境構築/Vue.js_Rails_Docker/)

## 2、Vue.Draggableをインストール
``` bash
npm install vuedraggable --save-dev
```
## 3、下記を`App.vue`に書き込む
``` javascript
import draggable from 'vuedraggable'
```
